# pry-byebug

To install `pry-byebug`:

```
gem install pry-byebug
```

To use it, require it:

```ruby
require "pry-byebug"
```

To insert a breakpoint:

```ruby
binding.pry
```

To continue with the next breakpoint, type in the prompt:

```
next
```

or

```
continue
```

To exit, use `exit`, `quit` or `!!!`
